Retail Sales Dashboard (Power BI)
Project Overview

This project is an interactive Retail Sales Dashboard developed using Power BI to analyze sales performance and provide business insights. The dashboard enables management to monitor revenue, transactions, customer activity, profitability, and regional performance through interactive visualizations.

Tools Used
Power BI Desktop
Power Query
DAX
Excel
Data Model

The project follows a Star Schema consisting of:

Sales Fact Table
Customer Dimension
Product Dimension
Store Dimension
Employee Dimension
Calendar Dimension
Dashboard Features
Executive KPI Cards
Total Revenue
Total Transactions
Average Order Value
Unique Customers
Profit Margin
Profit Trend Analysis
State-wise Transaction Map
Category-wise Analysis
Department-wise Analysis
Region-wise Analysis
Interactive Slicers
DAX Measures
Total Revenue
Total Transactions
Average Order Value
Total Profit
Profit Margin
Unique Customers
Business Insights

The dashboard helps management

Monitor overall sales performance
Identify high-performing regions
Compare product categories
Track profitability
Support data-driven business decisions
